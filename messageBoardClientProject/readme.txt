CS3201 Programming Assignment: MessageBoardClient
======================================================================================

Development environment:
OS       : macOS Sequoia 15.1
Language : Python 3.13.0
IDE      : Visual Studio 1.95.0 (Universal)
Shell    : zsh 5.9 (arm64-apple-darwin24.0)


Files included:
1. MessageBoardServer.py (given file)
2. MessageBoardClient.py (client program)
3. readme.txt 

How to run:
1. Ensure the system has python3
2. Run the MessageBoardServer on the correct IP address and port.
3. In the terminal or command prompt, go to the directory with 'MessageBoardClient.py'.
4. Run the client program with the command below:
    python3 ./messageboardclient.py <serverIP> <serverPort>
